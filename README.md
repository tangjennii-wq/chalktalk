# Chalk Talk Generator

A free, web-based tool that generates 10-minute board-aligned chalk talks for internal medicine doctors. Built by Jennifer Tang, MD (Nephrology, NYU Langone).

**Live tool:** https://tangjennii-wq.github.io/chalk-talk/

## What it does

- Generates lecture-style or board-style chalk talks across all internal medicine subspecialties
- Anchored to current society guidelines (KDIGO, AHA/ACC, ATS/ERS, ACG/AASLD, ADA, IDSA, ASH, ASCO, ACR, AAN, SCCM, AAAAI, AAD, APA, AGS, ACOG, USPSTF, AAHPM)
- Built-in chat for refining content
- Optional AI medical illustrations and podcast audio
- Peer-review pass for accuracy

## Requirements

Users supply their own Anthropic API key (free to create at console.anthropic.com). The key is stored locally in the browser only — never transmitted anywhere except Anthropic.
